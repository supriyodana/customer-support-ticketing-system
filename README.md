
## About

A frontend-focused ticketing system where tickets are stored in locally.
It includes status management, priority filters, comments, and dark mode, seeded sample data on first run.

## Includes

- Dashboard with ticket statistics
- Two viewing modes: card view and table view
- Search by ticket title
- Filter by status and priority
- Add ticket form with Formik + Yup validation
- Ticket preview modal before submit
- Ticket details page with:
- Status updates
- Comment thread
- Delete flow with confirmation modal
- Dark/light theme toggle with persisted preference
- Seeded demo tickets when storage is empty

## Tech Stack

- React 19
- Vite 7
- React Router DOM
- Tailwind CSS 4
- Ant Design + Ant Design Icons
- React Icons + Lucide React
- Formik + Yup
- ESLint
- GitHub Pages (deployment)
